# MVP対象外

以下は意図的にMVPへ含めない。Codexは仕様を拡大して勝手に実装しない。

- サーバサイド処理
- ユーザーアカウント
- オンラインランキング
- クラウドセーブ
- PvP
- 協力プレイ
- 課金
- 広告
- 不正防止
- 実績システム
- Prestige、転生、New Game+
- オフライン進行
- スマートフォンのネイティブアプリ
- PWAインストール
- 通知
- 効果音とBGM。余力があっても別PRとする
- 多言語化。内部文字列を過度に密結合させない程度でよい
- ユーザー作成問題エディタ
- 問題URL共有
- 唯一解保証
- 65セル以上の盤面
- ペントミノ等、4セル以外のピース
- ピースの鏡映操作
- ソルバ探索stackの永続保存
- 生成・プレイデータの外部テレメトリ送信
- 派手な3D、Canvas、WebGL
- UIライブラリ導入による大規模なデザインシステム

MVP完成後に検討する拡張候補は、Prestige、異形盤面、問題エディタ、オフライン進行、より高度なExact Cover実装、ソルバ同士のアルゴリズム比較である。
